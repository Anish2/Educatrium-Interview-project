Educatrium Interview Project
Author: Anish Visaria <anishvisaria98@gmail.com>

Technologies Used: Django, React.js, MySQL DB

How to use:
--------------------------------
Start server with command: "python manage.py runserver 8000"

Project will be at: "http://localhost:8000/interview_project/"



Note: there will be no data to start with so it can be made manually at "http://localhost:8000/admin/" or generated at "http://localhost:8000/interview_project/generate"
